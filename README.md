JEEAngularCRUD
==============

Sample CRUD Application shows a JEE REST Backend and a Angular HTML5 Client (Developed in Netbeans 8)

Installation
============

CRUDClient
----------
	- npm install
	- bower install
	- grunt serve / or use Netbeans Chrome Extension

CRUDServer
----------
	- mvn clean install
	- Project can be direct started in Netbeans 8 (Glassfish)
