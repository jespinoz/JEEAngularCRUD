JEE Angular CRUD.

This Sample CRUD Application shows a JEE7 REST Backend. The REST Service Backend is generated by the Netbeans REST webservice from database wizzard. The REST backend implements a CRUD interface to the customer table of the sample derby database. The Client is a single page HTML5 application implemented with AngularJS. The AngularJS Code is partial generated by the Yeoman Angular Generator.

Build on the command line. All tasks could be also done inside Netbeans. 

$ cd CRUDClient
$ npm install
$ bower install

$ cd CRUDServer
$ mvn install
