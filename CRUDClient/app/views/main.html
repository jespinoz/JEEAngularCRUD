Customer CRUD Demo

This demo shows a combination of a JEE7 REST Backend with a Angular HTML5 Single Page Application.

It is a basic CRUD interface to the customer table of the derby sample database. The sample database is included in the Netbeans 8 JEE Edition.

Components

This sample uses the following frameworks / components

  • REST Server
    • Netbeans 7
    • Glassfish 4
    • JAXRS
    • JPA
  • HTML5 Client
    • Netbeans 7
    • Angular JS
    • Yeoman - YO, Grunt, Bower
    • Bootstrap CSS

  • Benefits
    • Stateless server provide the best scalability.
    • Reusable REST service can support multiple frontend channels

  • Drawbacks
    • Developers which implements both Tiers have to learn multiple technologies
    • Only a small part of the client technlogy is standarized.
